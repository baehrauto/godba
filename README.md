# DBA Logistics Website

Modern, responsive website for Distribution By Air (DBA) - a logistics and shipping solutions company.

## Features

- **Modern Design**: Clean, professional UI with smooth animations and transitions
- **Video Background**: YouTube video background in hero section with gradient overlay
- **Mega Menu Navigation**: Comprehensive multi-level navigation with all services and industries
- **Interactive Industries Section**: Hover over industries to view related images
- **Responsive Design**: Fully responsive across all device sizes
- **Contact Forms**: Integrated contact and quote request forms

## Technologies

- HTML5
- Tailwind CSS
- JavaScript
- Google Fonts (Inter & Poppins)

## Pages

- Home (index.html)
- Services (Domestic, International, Specialized, etc.)
- Industries (Healthcare, Automotive, Military, etc.)
- Customer Tools (Track Order, Make Payment, Request Quote)
- About Us, Careers, Testimonials, Leadership Team
- Contact Us

## Setup

1. Open `index.html` in a web browser
2. Or serve locally using a simple HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Structure

- `index.html` - Main homepage
- Individual HTML pages for each service, industry, and section
- All pages share the same navigation structure and footer

## Notes

- Contact information uses placeholder/generic data
- YouTube video background requires internet connection
- All images are loaded from Unsplash CDN
